🚀 GET STARTED IN 5 MINUTES

1️⃣  Open PowerShell
    cd C:\Users\ojasw\OneDrive\Desktop\AI_Anomaly_LDR

2️⃣  Setup environment
    python -m venv venv
    venv\Scripts\activate
    pip install -r requirements.txt

3️⃣  Train model
    cd src\ml
    python train_model.py

4️⃣  Start backend (keep open)
    cd ..\backend
    python -m uvicorn main:app --reload

5️⃣  Start frontend (new terminal, keep open)
    cd ..\frontend
    npm install
    npm run dev

6️⃣  Generate test data (new terminal)
    cd ..\ml
    python -c "
    import requests, time
    from data_simulator import LDRSimulator
    sim = LDRSimulator()
    for r in sim.generate_normal(30):
        requests.post('http://localhost:8000/ingest-data', json=r)
        time.sleep(1)
    "

7️⃣  Open dashboard
    http://localhost:5173
